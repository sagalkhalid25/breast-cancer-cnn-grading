Breast Cancer Histopathology: Patch-Based Deep Learning Pipeline

This project presents a clean and reproducible deep learning pipeline for breast cancer tumour grade classification using digital histopathology images from the TCGA-BRCA dataset. The workflow applies patch-based convolutional neural networks (CNNs), slide-aware validation, and interpretability techniques to support transparent and auditable model evaluation.

The pipeline compares ResNet-34 and EfficientNet-B0 backbones using a strict leave-one-slide-out (LOSO) cross-validation strategy to avoid data leakage and over-optimistic performance estimates. Grad-CAM visualisations highlight model attention within tumour-rich regions.

üî¨ Pipeline Summary

  • Whole-slide image (WSI) processing and patch extraction
  • Weak slide-level labels based on Nottingham grading
  • Balanced index generation and LOSO evaluation
  • Training of ResNet-34 and EfficientNet-B0 backbones
  • Grad-CAM heatmaps for spatial interpretability
  • Fully reproducible environment with fixed seeds
