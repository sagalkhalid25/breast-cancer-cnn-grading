Breast Cancer Histopathology: Patch-Based Deep Learning Pipeline

Overview

This project presents a clean and reproducible deep learning pipeline for breast cancer tumour grade classification using digital histopathology images from the TCGA-BRCA dataset. The workflow applies patch-based convolutional neural networks (CNNs), slide-aware validation, and interpretability techniques to support transparent and auditable model evaluation.

The pipeline compares ResNet-34 and EfficientNet-B0 backbones under strict leave-one-slide-out cross-validation to avoid data leakage and over-optimistic performance. Grad-CAM visualisations provide insight into spatial model attention.

Pipeline Summary

  • Whole-slide image (WSI) processing and patch extraction
  • Slide-level weak labels (Nottingham grades)
  • Balanced index generation and LOSO validation
  • Training and evaluation of ResNet-34 and EfficientNet-B0
  • Grad-CAM heatmaps for interpretability
  • Fully reproducible environment with fixed random seeds
